1. 系統概述與適用情境

ACHB RFQ 估價系統是一套以瀏覽器執行的互動式報價工具，不需安裝任何軟體，直接開啟 rfq-estimate.html 即可使用。

兩種估價模式

模式適用情境備註
單工件
Single		 客戶詢問單一料號；標準精密加工零件 最常用模式，涵蓋材料、廠內加工、廠外製程、外購物料
組合件
Assembly		 多個工件組成一個組裝 set；複雜產品報價 可設定每個工件的個別材料與製程，共用外購物料

估價輸出項目

  • 各數量級距下：材料、廠內加工、廠外製程、外購物料、設定費、一次性費的分項成本
  • 建議售價（USD / pc 或 USD / set），含毛利率
  • 各級距總金額一覽表
  • 可匯出 Excel 備查、或儲存 JSON 以便日後載入修改

2. 估價前置準備

開始前請準備：
  • 客戶提供的 RFQ 文件（料號、版次、材料規格、表面處理要求）
  • 工程師評估的各 OP 加工時間（分鐘 / 件）
  • 材料供應商報價（含數量級距）
  • 廠外製程廠商報價（含數量級距）
  • 當期 USD/NTD 匯率

開啟工具

直接用 Chrome / Edge 開啟 rfq-estimate.html。不需網路連線，所有計算均在本機執行。

注意：工具資料只存在瀏覽器記憶體中，關閉頁面即清除。估價完成後務必「儲存 JSON」或「匯出 Excel」留底。

3. 全局參數設定

頂部工具列為全域參數，影響所有製程的計算，請在填入製程資料之前先確認這些設定。

參數說明常用值 / 注意事項
匯率（NTD/USD） 廠內加工設定費（USD）換算為 NTD 的匯率 預設 30。請依當日銀行賣出匯率調整。
設定費模式 決定使用哪個設定費費率 量產：一般交期訂單

5. 設定數量級距

預設已有 5、50、100 三個級距。根據客戶詢價的數量需求自行新增或刪除。

1

刪除不需要的預設數量

點擊數量標籤右側的「×」移除。

2

新增所需數量

在下方輸入框填入數量（整數），點「＋ 新增」。

建議設定 3～5 個有代表性的級距，例如：50、150、500。

計算邏輯：設定費、一次性費用（刀具/治具/模具）皆依各級距數量均攤。數量越多，每件攤提費越低，售價通常隨之降低。

6. 單工件模式操作流程

確認頂部 Mode 切換為「單工件」（預設）。依序填入以下區塊。

6.0 同張 RFQ 多工件評估 V3.0 新增

當客戶在同一張 RFQ 中詢問多個料號時，可在單工件模式下開啟多個工件 tab，每個 tab 獨立估算成本，共用同一份匯率、毛利設定與數量級距。

1

新增工件 Tab

頁面最上方「工件列表」列中，點「＋ 新增工件」即可新增一個工件 tab。預設顯示 Part 1、Part 2… 待填入料號後，tab 標籤會自動更新為該料號。

2

切換工件

點擊 tab 按鈕即可切換至該工件的輸入欄位，右側結果面板同步更新為該工件的估價結果。
每個工件各自有獨立的：料號、材料、製程、外購物料、一次性費用。

3

查看各工件總覽

右側結果面板底部會顯示「各工件售價總覽」表，列出所有工件在各數量級距下的建議售價（USD/pc），方便一次比較。

4

刪除工件

點 tab 按鈕上的「×」即可刪除該工件（至少保留一個 tab，無法刪至 0）。

5

匯出 Excel

多工件時，點「⊞ 匯出 Excel」會為每個有填製程的工件建立獨立工作表，並額外生成「總覽 Summary」工作表，顯示所有工件的跨級距售價。

共用設定：匯率、難度/毛利、佣金、數量級距為所有工件共用。RFQ 單號與客戶名稱也是共用欄位，請在切換 tab 前先填好。

6.1 材料成本

1

新增材料價格級距

點「＋ 新增價格級距」。每個級距需填：

  • 數量（pcs）：達到此數量以上時採用該單價。留空代表不限數量固定單價。
  • 單價（NTD/pc）：含稅或不含稅請統一口徑。

只有一個固定單價時，數量欄位留空即可。

2

填入架機數量（選填）

若訂單量與實際備料數量不同（例如：訂 50 件但需備料 100 件方可開機），請填入架機數量。

系統會以架機數量查對材料級距，但仍按訂單量分攤成本至每件。

若無此情況，留空即可（自動以訂單量查對）。

6.2 製程途層（Process Routing）

每個製程步驟（OP）分為兩種類型：

廠內 廠內機台加工，按機台費率 × 加工時間計算成本，加工時間有設定費
廠外 委外廠商報價，直接輸入廠商每件報價（含數量級距）

1

點「＋ 新增製程步驟」

預設新增廠內製程；點擊步驟頭部的「廠內 / 廠外」切換鈕可更改類型。

2

廠內製程：填入三個欄位

  • 機台 / 製程：從下拉選單選擇對應機台（含費率提示）
  • 加工時間（min/pc）：每件的加工時間，單位分鐘
  • 設定時間 Setup（hr）：每次開機的設定工時，單位小時（量產或首次製作依頂部設定決定費率）

步驟標題列會即時顯示：機台名稱 · 時間 · 廠內加工成本 / 件。

3

廠外製程：填入廠商與報價

  • 製程名稱：輸入或從建議清單選擇（如「陽極 Anodizing」）
  • 廠商名稱：記錄報價廠商
  • 數量級距報價：每個級距填入達到該數量時的 NTD/pc 報價；只有一個固定報價時，數量留空即可
4

重複新增直到所有 OP 填完

步驟可展開 / 折疊（點擊標題列）。完成的步驟建議折疊以保持畫面整潔。

可選機台一覽

機台群組範例機台費率（NTD/min）
走心 CNCCNC1、CNC5、CNC9、CNC18、CNC217 ～ 15
走刀 CNCCNC11～17、CNC19、CNC2210.5 ～ 15
MC / 加工中心MC加工鋁、MC加工白鐵、MC加工塑膠、臥式11 ～ 16

  • 走心 / 走刀 CNC：設定費 USD$45/hr（一般機台）或 USD$50/hr（高階機台）
  • MC / 臥式：設定費 USD$50/hr
  • 研磨、雷刻、後製程：無設定費（費率已含固定成本）
  • 設定費依「設定費模式」（量產 vs 首次製作）採用對應的工時投入

6.3 外購物料清單（Purchased Parts）

指需要向外部採購、直接用於此零件的標準品（如 O-Ring、螺絲、墊片等）。

1

點「＋ 新增品項」

每筆填入：品項名稱、單價（NTD）、用量/件。

2

選擇計算方式

  • 固定/件：用量 × 單價 直接加入每件成本（最常用）
  • 批次攤提：用量 × 單價 為批次總費用，再除以當前級距數量攤提。
    適用於整批採購但可跨多個訂單使用的耗材。

6.4 刀具 / 治具 / 一次性費用

指每次新料號或換版需要的一次性投入費用。系統依各數量級距均攤至每件成本。

費用項目說明
刀具費 Tooling特殊刀具、成形刀等首次費用
治具費 Fixture夾具、定位治具的製作費
模具費 Mold射出、壓鑄等模具費用
其他 Others其他一次性費用（包材、遮噴工裝等）
攤提方式：一次性費用 ÷ 級距數量 = 每件攤提成本。
例：刀具費 NT$10,000，訂單量 50 件 → 每件攤 NT$200；訂單量 500 件 → 每件攤 NT$20。

7. 組合件模式操作流程

點擊頂部 Mode 切換為「組合件」。組合件模式下，每個工件（Workpiece）獨立設定材料與製程，最後加總為整組（set）的售價。

1

新增工件

點「＋ 新增工件」，系統自動建立 WP01、WP02…

每個工件填入：名稱、料號、材料，以及每組用量（pcs/set）。

2

為每個工件設定材料成本與製程

操作方式與單工件模式相同（見第 6.1、6.2 節）。

工件可展開 / 折疊，點擊工件標題列控制。

3

填入組合件共用外購料

在頁面最下方「組合件外購料（共用）」區塊，填入跨工件共用的購買零件（如螺栓套件、封裝袋等）。

4

確認每件用量

每個工件標題列的「每組用量（pcs/set）」是關鍵。若某工件每組需 2 個，請填 2；系統會將成本 × 2 後納入組成本。

組合件結果側邊欄會額外顯示「工件成本拆解」，讓你看到各工件佔整組成本的比例，方便判斷哪個工件是成本主體。

8. 結果解讀

右側即時估價欄位在每次修改後自動更新。

數量頁籤切換

點擊頂部數量頁籤（如 50 pcs / 150 pcs…）切換目前查看的級距，詳細成本明細隨之更新。

成本視覺化長條

顏色對應：藍色 — 材料 綠色 — 廠內加工 紫色 — 廠外製程 橙色 — 外購+攤提

成本明細表解讀

欄位說明
材料成本 Material依架機數量查對級距單價後換算的每件材料費
外購物料 Purchased Parts所有外購品項計算後的每件合計
廠內加工 In-house所有廠內 OP 的機台費率 × 時間加總（不含設定費）
廠外製程 Outsourced所有廠外 OP 的廠商報價加總（依級距取對應報價）
設定費攤提 Setup Amort.設定費（設定工時 × 設定費率 × 匯率）÷ 訂單數量
一次性費攤提 Tooling Amort.（刀具 + 治具 + 模具 + 其他）÷ 訂單數量

各級距售價總覽

頁面底部的 All-Tier Summary 表格一次顯示所有數量級距的成本與售價，可直接截圖貼入報價郵件。

建議售價為參考值。業務得依客戶關係、競爭報價、市場行情調整最終報出的價格。此工具計算的是成本導向下限，不替代商業判斷。

9. 匯出與儲存

儲存 JSON（↓ 儲存）

將當前所有輸入資料存為 .json 檔案至本機。日後可透過「↑ 載入」恢復，繼續修改估價。

建議命名規則：RFQ-[料號]-[版次]-[日期].json
例：RFQ-M10848-RevD-20260524.json

匯出 Excel（⊞ 匯出 Excel）

產生 .xlsx 檔案，包含：

  • 各級距成本明細工作表（可供主管審核）
  • 所有輸入參數（料號、製程設定等）
Excel 用途：留存報價紀錄、併入 RFQ 彙整表一起存檔。

清空（✕ 清空）

清除所有輸入資料，回到初始狀態。操作後無法復原，請先確認已儲存。

10. 計算公式參考

廠內加工成本（每件）

廠內成本 = Σ ( 各 OP 加工時間(min) × 機台費率(NTD/min) )

設定費攤提（每件）

設定費攤提 = Σ ( 設定工時(hr) × 設定費率(USD/hr) ) × 匯率(NTD/USD) ÷ 訂單數量(pcs)

一次性費用攤提（每件）

一次性攤提 = ( 刀具費 + 治具費 + 模具費 + 其他 ) ÷ 訂單數量

總成本（每件）

總成本 = 材料成本 + 外購物料 + 廠內加工 + 廠外製程 + 設定費攤提 + 一次性攤提 [ + 佣金(5%) ]

建議售價（USD / pc）

建議售價(USD) = ( 總成本(NTD) ÷ 匯率 ) ÷ ( 1 − 毛利率 ) 例：總成本 NTD 600，匯率 30，毛利率 30% → 600 / 30 = USD 20（成本） → 20 / 0.7 = USD 28.57（建議售價）

廠外製程計價原則

廠外製程報價直接加入總成本 → 再用 (1 − 毛利率) 倒算售價 等效：廠外報價 / 0.7 ≈ 此項製程對售價的貢獻（30% 毛利時）

11. 常見問題排查

現象可能原因解決方式
側邊欄顯示「尚未輸入製程資料」 尚未新增任何製程步驟 點「＋ 新增製程步驟」至少加入一個 OP
廠內加工成本為 0 加工時間（min/pc）欄位為空或 0 填入正確的每件加工時間
設定費攤提為 0 設定時間（hr）欄位為空或 0，或所選機台無設定費（研磨、雷刻等） 確認機台費率說明（綠色提示欄）；研磨類機台設定費設計為 0
廠外報價未在不同級距顯示差異 廠外報價只填了一個無數量限制的固定價 點「＋ 新增報價級距」，分別填入不同數量對應的 NTD/pc
材料成本在不同級距相同 材料只有一個不限數量的固定單價 若有量大優惠，新增多個材料價格級距
載入 JSON 後數量顯示異常 JSON 檔案由不同版本工具產生 手動確認各欄位資料後重新計算；必要時手動重填
匯出 Excel 失敗 瀏覽器安全性封鎖檔案下載 確認瀏覽器允許此頁面下載；建議使用 Chrome 或 Edge
重要提醒：關閉或重整頁面會清除所有未儲存的資料。養成每完成一個 RFQ 就立即「↓ 儲存」JSON 的習慣。
