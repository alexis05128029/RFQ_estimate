V3.0 2026-05-25
  • FEAT
    多工件 Tab 系統：單工件模式新增工件列表 Tab Bar，同一張 RFQ 可開啟多個工件 Tab 獨立估算，共用匯率 / 毛利 / 數量級距設定。Tab 標籤自動帶入 Part No。
  • FEAT
    各工件售價總覽：開啟多個工件時，右側結果面板底部自動顯示所有工件在各數量級距下的建議售價對照表（Multi-Part USD/pc Summary）。
  • FEAT
    Excel 多工件匯出：多 Tab 時，匯出 Excel 為每個有製程的工件建立獨立 Sheet，並額外生成「總覽 Summary」工作表彙總各工件跨級距售價。
  • CHG
    State 架構重構：全域參數（匯率 / 毛利 / 數量級距）獨立為 G 物件；每個工件的料號、材料、製程、外購料、一次性費用各自存在 Tab State，切換 Tab 時完整還原。
  • CHG
    儲存格式升版 v2：JSON 儲存格式升至 version:2，包含完整 TABS 陣列與 G 全域設定；向下相容 v1 舊檔（自動轉換為單 Tab）。
  • FIX
    一次性費用（刀具 / 治具 / 模具 / 其他）改為 per-Tab 儲存，切換工件時正確還原各工件的獨立費用設定。
